# CentOS7命令

命令列表

1. `cut`: 

   > Print selected parts of lines from eachc FILE to standard output.

2. `grep`:

   > 在文件或输入中查找pattern，pattern是基本正则表达式。

3. `cat`:

   > 将文件或输入组合输出到标准输出。

4. `tac`:

   > 与cat相同，但是倒着输出。

5. `nl`:

   > 带行号输出。

6. `pwd`:

   > 是 Print Working Directory （打印工作目录）的缩写，功能是显示用户当前所处的工作目录。

7. `whoami`:

   > 用于确定当前登陆的用户。

8. `tail`:

   > 输出文件的末尾部分。
