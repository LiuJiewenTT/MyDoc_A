## Mine

原本是不支持父类选择器的，即便是CSS3。但现在已经支持了(2023)，只是有些浏览器还未支持。

目前，桌面端谷歌、edge、opera、safari已支持，火狐需打开开关；移动端，谷歌、opera、safari、Android已支持。
